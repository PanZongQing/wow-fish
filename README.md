## WOW 自动钓鱼

### 原理
通过图片匹配来查找鱼漂位置, 通过鼠标指针变化来判断是否找到鱼漂, 通过水花图片匹配来确定是否有鱼上钩

### 配置
* 鱼漂采样图片放在float文件夹中,可以有多张,但名称应该按照顺序来,不能跳过
* 水花采样图片放在splash文件夹中, 其他同上
* 目前只在奥伯丁测试过,其他地方请自行采样
* 其他配置均在main.py中,请参照注释,比如钓鱼快捷键设置等

### 运行方式
python main.py
